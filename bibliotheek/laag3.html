Code

Laag 3: CODE
Titel kruimelspoor: De Maskerade van de Betekenis — hoe de Code ons betovert
Startinzicht:
“De wereld is getekend in tekens. Maar wie de klank vergeet, leest nooit de waarheid.”

Kerninhoud:
In deze laag onthul je hoe code — in taal, typografie, typologie en zelfs in wetten — niet neutraal is, maar een systeem op zich. De code doet werk. Ze doet alsof ze betekenis is, maar is vaak juist de maskering van intentie.

- Fonetisch alfabet → klank vóór betekenis (GOES = GOOSE)
- Typografie van VEERE ↔ Abraxas-zegels → vorm = boodschap
- “Ot en Sien” → indoctrinatie via kinderbeeld & klank
- Sourcy → flesje verlossing i.p.v. veld
- “Not” = ver-antwoordelijkheid, taal als afleiding

Bewijsmateriaal (flarden en beelden):
- Davitamon / Sourcy / vitaminewater
- Wide World Game (1933!) → verborgen wereldscript
- Hanenmasker Burkina Faso → masker als code
- Poster met VEERE-typografie + Abraxas
- Joernael Groenlants-Vaerders (Zeeusche Veer)
- Enoch-manuscript → gecodeerde mythen
- Unicornus uit Bartholini

Essentie:
Code is de sluier. Een zorgvuldig geknutselde illusie, die je alleen ziet als je hoort. Letter = lijst. Klank = leven.
